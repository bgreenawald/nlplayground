Games

This area is for any interest web based games that can in some way be related to text processing. The only game currently available is application that can take an phrase and "Mad Gab"-ify it (Google "Mad Gab" if you aren't sure what that means). This uses a custom algorithm that I authored to take any phrase and return it's "Mad Gabed" version. I intend to make a blog post about the algorithm soon. A friend and I are also working on an online version of Mad Gab that utilizes the algorithm.
