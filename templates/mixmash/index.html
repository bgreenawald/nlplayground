Markov Mix and Mash

This is a set of word-level models inspired by the Random Kanye project. Using Markov chains, we can create models that combine the text from two or more sources and can output new text that mixes the sources. Because it uses Markov chains, the models do not have a good sense of memory, meaning the output text generally will make sense every couple of words, but will overall be gibberish.
